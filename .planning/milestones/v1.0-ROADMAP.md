# Milestone v1.0: digitalist.se MVP

**Status:** SHIPPED 2026-01-13
**Phases:** 1-4

## Overview

Fran tom katalog till konverteringsdriven foretagssajt for suveran AI i offentlig sektor. Bygger foundation med ratt teknikstack, implementerar karninnehall med de fyra dimensionerna, adderar konverteringsoptimerat innehall, och avslutar med SEO/tillganglighet infor launch.

## Milestone Summary

**Decimal Phases:** None

**Key Decisions:**

- Next.js 15 (not 14) — security vulnerabilities in 14.2.x
- next-intl for i18n — type-safe, App Router native
- Directus 11.x — latest stable, self-hosted
- Three-palette colors — Primary (navy), Accent (blue), Concrete (gray)
- CSS clip-path icons — no external icon libraries
- Mock data pattern — CMS-ready but works without API
- Native Metadata API — no third-party SEO libs
- Focus trap via Tab interception — no external a11y library
- JsonLd component — reusable for all Schema.org types
- Article type for case studies — better than deprecated CaseStudy type

**Issues Resolved:** None encountered

**Issues Deferred:** None

**Technical Debt Incurred:**

- Placeholder client logos (text labels instead of actual images)
- Mock data instead of real Directus CMS content
- Console logging for form submissions (no CRM integration yet)
